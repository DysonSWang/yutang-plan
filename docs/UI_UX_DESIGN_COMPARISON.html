动效规范

统一 spring physics，使用 framer-motion 最佳实践

当前设计
// 分散的动效定义 transition: { duration: { 'ultra-fast': '150ms', 'fast': '200ms', 'normal': '300ms', }, easing: { 'ease-out-expo': 'cubic-bezier(0.16, 1, 0.3, 1)', 'spring': 'cubic-bezier(0.34, 1.56, 0.64, 1)', } } // 组件内直接使用 CSS _hover: { transform: 'translateY(-3px)', boxShadow: '0 8px 32px rgba(0,0,0,0.5)', },
!

framer-motion v6 不兼容 React 19

需要升级到 v11+ 或迁移到 CSS 动画

!

缺乏统一规范

每个组件各自定义动效，难以维护

优化方案
// 统一动效 hooks import { useSpring, useTransform } from 'framer-motion'; // 定义标准 spring const SPRING = { type: "spring", stiffness: 100, damping: 20, }; // 按钮动效组件 function MagneticButton({ children }) { const x = useSpring(0, SPRING); const y = useSpring(0, SPRING); return ( <motion.button style={{ x, y }} whileHover={{ scale: 1.02 }} whileTap={{ scale: 0.98 }} > {children} </motion.button> ); }
动效规范
spring: stiffness 100, damping 20
过渡: cubic-bezier(0.16, 1, 0.3, 1)
入场动画: fadeSlideUp, staggerChildren 0.1s

优化方案
// 全局 focus-visible 样式 ':focus-visible': { outline: '2px solid var(--gold)', outlineOffset: '2px', }, // 按钮组件增强 <Button bgGradient="linear(...)" className="focus-ring" aria-label="登录" role="button" tabIndex=0 _focusVisible={{ boxShadow: "0 0 0 3px var(--gold-glow)", transform: "scale(1.02)", }} /> // 颜色对比度规范 // 主要文字: #f5f0e8 on #111110 = 14.5:1 ✓ // 次要文字: #a38e5a on #111110 = 5.2:1 ✓ // 禁用文字: #6b7280 on #111110 = 3.2:1 (需提升)
