## 一、现状与目标

## 六、安全设计

### 6.1 文件访问安全

- skills/ 目录放在 `src/coaches/` 下
- 用户无法通过URL直接访问
- 只通过 loader.js 读取

### 6.2 路径遍历防护

```javascript
function safeLoadSkill(skillId) {
  if (!validIds.includes(skillId)) return null;
  const filePath = path.join(SKILL_DIR, skillInfo.file);
  if (!filePath.startsWith(SKILL_DIR)) return null;  // 防护
  return loadSkill(skillId);
}
```

### 6.3 Prompt注入防护

```javascript
function sanitizeUserInput(input) {
  return input
    .replace(/\[.*?\]/g, '')           // 移除[SYSTEM]等标记
    .replace(/^(你是一个|你是).*?(：|:)/, '')
    .replace(/<\/?[^>]+>/g, '')        // 移除HTML
    .substring(0, 2000);               // 限制长度
}
```
