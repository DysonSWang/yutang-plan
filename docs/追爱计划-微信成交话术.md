# 缘分计划 · 微信成交话术

---

## 背景

```
客户已经看到了成果（你带的女生让他震撼）
他主动说"搞起来"、"让大家的缘分里充满美人鱼"
他已经心动，等你揭晓答案
```

---

## 核心原则

```
1. 先说结果，后说价格
2. 锚定商K年消费 + 商务陪伴
3. 具体化交付，不是说虚的"妹子自由"
4. 让他算账，不替他算
5. 他问才答，不主动解释
```

---

## 完整对话

```
你：琢磨好了😊 一个系统性的方案 帮你补齐妹子自由

客户：说说看？

你：5年经验，200多次约会实战。不是帮你找1个，是帮你建一个缘分体系。

客户：具体呢？

你：不是帮你找1个，是帮你建一个持续有鱼的圈层。1个月打造好，50+女生。前3个月每月2-3次约会。6个月找到愿意长期跟你的。1年后稳定运转，你随时约、随时选。

客户：具体怎么弄？

你：5万一年，分阶段。
先付50%启动，建好池子第二步，约会按月付，锁定长期关系第四步。
每个阶段你验收了，再给下一步。

客户：这么爽？不用我自己聊？

你：你的账号、你的圈子、你的感觉。我帮你运营，你负责翻牌子。你需要出面的时候告诉你，不需要你操心聊什么。

客户：那一年多少钱？

你：5万一年的朋友价。商K你一年20几万，每次打水漂。我5万帮你建圈层，持续有鱼。你说值不值？

客户：能不能便宜点？

你：不是便宜不便宜的问题。商K一年20几万，我5万帮你一年，建好池子，每月约会，长期有人。你自己算。

客户：行，那怎么弄？

你：我一个月只看3个，保证质量。下周给你开账号，先聊着，感觉有了再见面上战场。

客户：好，下周约个时间？
```

## 异议处理

### "能不能便宜点"

```
❌ 错误：你算算值不值
✅ 正确：不是便宜不便宜的问题。商K一年20几万，我5万帮你一年，建好池子，每月约会，长期有人。你觉得是那边划算，还是我这里划算？

逻辑：让他自己选择，不是让他砍价
```

### "万一没效果怎么办"

```
✅ 正确：分阶段付费。你看到东西了，再给下一步的钱。我有把握做成，才敢这么收。

逻辑：展示自信，不是给自己留退路
```

### "我要想想"

```
✅ 正确：想什么呢？昨天那个感觉，不想要了？我等你。

逻辑：直接逼单，不给他拖延借口
```

## 注意事项

```
1. 不要主动说分阶段——等他问再说
2. 不要展示成本——他买的是结果，不是你的时间
3. 不要说虚的"妹子自由"——要说具体交付
4. 不要解释太多——他问什么答什么
5. 不要显得心虚——你是帮他，不是求他
```
