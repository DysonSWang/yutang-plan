## 以后改了 seed 数据怎么发布

每次改了 `seed-chapters.js` 里的章节内容后，在服务器上执行：

```bash
cd /root/zhuiai/backend && git pull && npm run db:seed-force && pm2 restart zhuiai-backend
```
