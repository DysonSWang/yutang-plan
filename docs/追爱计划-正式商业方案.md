## 五、获客路径

### 陈辉民原则：优先级第一

> 「一个企业要增长，一定是业绩的增长。只有把找客户这件事的优先级提到前面来，才能够破局。」

### 你的优势

```
你有50人私域，粉丝信任你是"泡妞专家"
这是最大的资产，直接跳过建立信任的阶段
```

### 三步走策略

```
第一步：激活私域（现在）
├→ 发朋友圈测试，发红包激活
├→ 999元/人的内测价，限量50个
└→ 找到第一批愿意付费的人

第二步：跑通路径（1-2个月）
├→ 有人付费了，证明产品有价值
├→ 收集好评，建立社交证明
└→ 优化话术和交付流程

第三步：放大渠道（3-6个月）
├→ 达人分销（找情感类博主合作）
├→ 内容引流（抖音/小红书发内容）
└→ 私域裂变（老带新，给奖励）
```

### 推荐裂变机制

```
第二个月起，好用可以续费，或者推荐新用户来抵扣

推荐规则：
├→ 推荐1个新用户付费 → 获得500积分
├→ 推荐2个新用户付费 → 获得1000积分
└→ 以此类推，上不封顶

积分用途：
├→ 积分可100%抵扣续费（999元 = 999积分 = 续费1个月）
├→ 终身有效，无过期时间
└→ 老用户推荐越多，续费越便宜

示例：
├→ 推荐2个朋友付费 → 获得1000积分 → 续费只需0元
├→ 推荐5个朋友付费 → 获得2500积分 → 未来25个月免费用
└→ 推荐10个朋友付费 → 获得5000积分 → 未来5个月免费用
```

> 陈辉民说：「老客户带新客户，成本接近零，但需要设计激励机制，不能靠人情。」

## 六、成交SOP

### 陈辉民成交四步

```
第一步：筛选（购买力 → 意愿度 → 认可度）
    ↓ 不是问要不要买，而是证明他符合条件

第二步：价值点（限时优惠、额外服务）
    ↓ 让客户觉得现在买最划算

第三步：成交（直接给出选择，不要问要不要）
    ↓ 「你现在可以付款了」

第四步：追付款（24小时内跟进）
    ↓ 付款了才叫成交
```
