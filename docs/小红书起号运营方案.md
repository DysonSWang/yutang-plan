## 二、内容策略

### 核心格式：诊断 + 话术 + 思维

**升级自**："青铜/白银/王者"话术对比

**新格式**：
```
问题场景 → 诊断先行 → 话术展示 → 思维提炼 → 行动触发
```

**为什么升级**：
- 话术展示"怎么回"，思维展示"为什么这样回"
- 用户不只收藏话术，还能举一反三
- Mo哥从"话术库"升级为"情感诊断专家"

## 三、内容系列规划

### 系列1：《聊天诊断》（40%）

| 期数 | 场景问题 | 目的 |
|------|----------|------|
| 第1期 | "在干嘛"怎么回 | 入门，经典场景 |
| 第2期 | "你喜欢我什么"怎么回 | 废物测试化解 |
| 第3期 | "你谈过几个"怎么回 | 挖坑题怎么答 |
| 第4期 | "你是我喜欢的类型"怎么回 | 表白类问题 |
| 第5期 | "约不出来怎么办"怎么回 | 操作类问题 |

**模板**：
```
【标题】：女生问"在干嘛"，王者从来不直接回答

【诊断先行】：
   青铜：在干嘛？（×）暴露需求感
   白银：刚下班，你在干嘛？（✓）有来有往，但太平淡
   王者：看心情 / 你猜？（✓✓）反客为主

【思维提炼】：
   → 为什么青铜是错的？不是因为话术，是因为暴露了需求感
   → 为什么王者是对的？因为掌握了主动权
   → 核心：回复 = 判断阶段 + 管理框架 + 控制需求感

【行动触发】：
   → 下次回"在干嘛"之前，先问自己：她在测试我还是真的关心我？
   → 评论区扣"思维"，送《聊天诊断手册》
```
