# 她说做朋友吧，怎么回？

---

## 发布文案（可复制粘贴）

她说"我们还是做朋友吧"。

你愣住了——「我哪里做错了？」「是不是不够好？」「我要不要挽留？」

Stop。把手从键盘上拿开。

我先坦白：当时收到朋友卡，发了一大段话挽留，她直接拉黑了。后来才懂——越追越死。

先说缺点：这套方法不是每次都灵。但如果她已经做了决定，沉默比追问好一百倍。

99%的男生收到朋友卡，第一反应就是追问、挽留、求情。但这三种反应，只会让她更确定：你是低位。

**先问自己一句：她善良吗？她值得你花这个精力吗？**

如果不值得，最好的回复就是沉默。如果值得，先搞清楚——她是哪种情况：

🔵 A型·真没窗口 — 回复越来越慢，约她永远没空，她已经做了决定
🔴 B型·在测试你 — 之前聊得不错，突然说做朋友，在观察你的反应
🟡 C型·朋友框架 — 你有价值但没吸引力，她需要你当免费司机/ATM

记住：她不会跟毫无价值的人说"做朋友"。这句话说明你还有用，但已经没有吸引力了。

**收到朋友卡，核心原因只有一个：她识别到了你的需求感。**

秒回消息、总主动找她、从不拒绝——你的需求感爆表了。吸引力 = 价值感 - 需求感，需求感一高，吸引力归零。

**正确应对：不要回答她的问题，让她来回答。**

❌ "为什么？我哪里做错了？" → 暴露需求感
❌ "给我一次机会" → 乞求姿态
❌ "好吧，那做朋友" → 坐实框架

✅ "谁跟你做朋友？我是男的" → 打破身份
✅ "好，那就先做朋友" → 不争辩，不暴露需求感
✅ "行，我先记上，反正我也没那么多时间" → 降低粘度，不围着她转
✅ "你这是在追我之前先发好人卡吗" → 反抛+调侃

记住：你是奖品，不是乞求者。

看完这7张图，下次收到朋友卡，你知道怎么做了。
