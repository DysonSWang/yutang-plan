# 鱼塘计划 · 项目说明

## 项目背景

帮创一代富男人（40-55岁，资产1000万+）找到真心喜欢他的心动女生的私密社交服务。

**一句话**：直接用客户的账号帮他全程聊，直到追到人。

**种子客户**：已有一个意向客户（有钱朋友），昨天看到了成果，正在洽谈中。

## 核心价值

帮富裕男人找到真心喜欢他的心动女生 - 不是约pao服务，是情感陪伴服务。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React + Chakra UI + Vite |
| 后端 | Express.js + Prisma ORM |
| 数据库 | SQLite |
| AI | 通义千问 (qwen3.6) + 智谱AI (glm-4) |

## 系统架构

```
frontend/          # React SPA (端口 5181)
  src/
    pages/
      admin/       # 操盘手端 (工作台、客户管理、女生资源、聊天、军师工具、进度)
      client/      # 客户端 (首页、档案、AI教练、我的鱼塘)
    components/    # 共享组件
    contexts/      # React Context (Auth, Socket)
    utils/         # API 封装
backend/           # Express API (端口 3005)
  src/
    routes/        # API 路由 (aiCoach.js, clients.js, girls.js, chat.js)
    services/      # 业务逻辑 (contextBuilder, signalExtractor, memory, learning)
    coaches/       # AI 教练配置 (configs/, skills/)
    middleware/    # 中间件 (auth)
prisma/             # 数据库 schema
```
