# 阅后即焚功能增强任务书

## 需求概述

将现有的「阅后即焚🔥」和「闪图✨」合并，简化为单一的阅后即焚功能，并增强配置选项。

### 功能规格

#### 1. 触发时机（二选一）
- **即时**：发送方点发送就启动倒计时，不管对方是否在线。时间到了服务器自动销毁消息，对方下次上线看不到。
- **阅后**：消息显示蒙层，对方点击查看后才启动倒计时。

#### 2. 时长选择（二选一）
- **固定秒数**：用户选择 3/5/10/15/30/60 秒
- **自适应**：所有类型统一为 5 秒（图片固定 5 秒，视频/音频按实际时长）

#### 3. 蒙层设计（阅后模式）
所有类型的阅后即焚消息在对方看到时都需要蒙层遮挡：
- **文字**：显示 `[阅后即焚]` + 橙色蒙层遮盖
- **图片**：缩略图 + 橙色蒙层，点击后打开查看器
- **视频**：缩略图 + 播放图标 + 橙色蒙层，点击后打开查看器
- **音频**：波形图标 + 橙色蒙层，点击后打开查看器

#### 4. 去掉的功能
- 删除 ✨ 闪图模式（`flashMode`、`isFlashImage`）
- 删除 FlashImageViewer 中的 `isFlashMode` 相关逻辑

const getMediaDuration = (file, type) => {
  return new Promise((resolve) => {
    const el = document.createElement(type === 'video' ? 'video' : 'audio');
    el.preload = 'metadata';
    el.onloadedmetadata = () => {
      URL.revokeObjectURL(el.src);
      resolve(el.duration);
    };
    el.src = URL.createObjectURL(file);
  });
};

## 注意事项

1. **sessionStorage 持久化**：倒计时信息存在 sessionStorage，刷新页面不丢失
2. **最低时长**：视频/音频自适应时长最低 3 秒
3. **蒙层点击**：只对接收方（`senderRole !== 'client'`）显示蒙层，发送方看到正常内容
4. **发送方行为**：发送方看到的是正常消息（无蒙层），倒计时在消息列表显示
