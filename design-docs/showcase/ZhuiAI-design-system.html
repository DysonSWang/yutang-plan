Design Philosophy

设计理念

「现代绅士」品牌人格 — 自信、干净、有温度的科技。不油腻、不廉价、不冷峻。暗色琥珀金体系，跨代际无争议的高级感。

绅士温度

琥珀金主色而非冷青。金色潜意识里是"值得信赖的成熟男性"——20岁觉得有品味，50岁觉得稳重可靠。

暗色克制

暖黑底色 + 哑光卡片。不做花哨渐变、不过度装饰。让用户聚焦于"人"本身——女生资料、聊天、约会。

移动原生

底部Tab导航、卡片流、拇指友好热区。目标用户在手机上使用这款私密服务，桌面为辅助场景。

建立信心

每像素传达"这个产品靠谱"。不卖焦虑、不搞PUA。视觉语言像一位靠谱的男性朋友——真诚、有品、不吹嘘。

情感链接

玫瑰铜作为情感色——约会标记、心动提示。与琥珀金形成冷暖双色体系，让产品有叙事层次。

渐进披露

首页展示核心入口，深度功能逐层展开。不让新用户面对过多选项，降低认知负荷。

Motion System

动效系统

克制、有品质。核心隐喻：余烬微光 (Ember) — 温暖、不刺眼、自然消散。所有动画时长 ≤ 400ms，使用物理缓动曲线。

品牌标识 · 余烬呼吸

品牌核心动效——琥珀金核心缓慢呼吸式缩放，三层同心圆向外扩散消失。像余烬的微光，持续、温暖、不喧闹。用于加载、空状态、品牌展示。
3200ms 缓入缓出循环 · 三层涟漪各偏 800ms

加载态 · 骨架屏流光

暖灰色渐变条 + shimmer 流光扫过。页面初始加载时使用，让用户感知内容即将到来。
1600ms 循环 · 暖灰 #252522 → #353530

页面入场 · 交错淡入上升

卡片依次入场，每个延迟 80ms。给用户"井然有序"的心理暗示。
550ms ease-out · 80ms stagger
